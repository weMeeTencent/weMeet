# WeMeet 微历

## 让相聚更容易

**微历**是一款聚焦多人组局、时间协调的工具型产品。

基于泛熟人社交的微信群聊，以小程序为产品形态，通过时间选择模式优化、实时呈现可视化统计结果、活动去处推荐一键走起、一键生成邀请函等产品功能，改善多人组局不便、时间难以协调、活动难以成行的痛点。
